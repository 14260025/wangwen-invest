## 一、投资哲学（道）

| 信条 | 要义 |
|------|------|
| 价值投资第一条件 | 伟大的祖国（国运）。国家好，假设才成立 |
| 元规则 | **供需关系是商业第一规则，一切服从供需关系** |
| 核心价值 | 垄断（自然垄断/资源品牌垄断/牌照垄断），能对抗通胀、产品能涨价 |
| 股票本质 | 股票=股权+彩票，能把握的只有股权；以矿主心态投资 |
| 风险定义 | 不能理解、不能把握的=风险；能理解把握的=机会 |
| 熊市防线 | "分红度一切苦厄"——高分红是下跌市最后防线 |
| 周期律 | 万物"成住坏空"；坏完了会变好，不会一直不好 |
| 心态 | 悲观者永远正确，乐观者才能赚到钱；对下跌高兴（段永平） |
| 修行 | 不"着相"（不局限于某行业）；长期复利15%即满意 |

## 二、五维选股：两高一低可持续 + 有梦想

1. **低估值**：估值给商业模式，分行业给区间；跌到下沿/跌破下轨才买。煤炭银行10倍PE可能贵，茅台25倍PE可能便宜。
2. **高现金流**：负现金流、画饼星辰大海的不投；只投能分享经营成果的阶段。
3. **高分红（中心）**：穿透三张表+检验管理层人品；熊市防线。
4. **业务长期可持续**：避减量市场（如钢铁虽符合前三点但不可持续）。
5. **有梦想**：前四点保不亏，此点是超额收益（预期差）来源。

## 三、供需判断行业周期（术·核心，详见 supply_demand_cycle.md）

**传导链**：股价 ← 价格 ← 供求关系 ← 供应端资本行为 ← 生产者/国家态度/投资

**买入**：供求关系逆转（供不应求）= 需求增+无新供应（资本进不来、头部不扩产、政策限产）→ 第一笔买，爆发集中仓。
**卖出（三红线）**：
1. 头部公司纷纷扩产（最危险）
2. 外源变故改变供需（让公司更不值钱就卖）
3. 找到明显更优标的（换仓集中）

**周期长度**：可达8年+，下行特别长（企业退出才运转）。底部有"十八层"，须确证供需逆转才重仓。
**上涨不均衡**：第一50%等三年，第二50%半年，第三50%三周，第四50%一周 → 反转即买、爆发加仓。

**四大周期**：经济/货币 · 行业 · 企业/产业 · 人性/情绪（全民谈股=顶，90%亏钱=底）。

**调研**：踏雪寻梅（下行期就关注）+ 交叉验证（董事长看愿景/工程师看技术/保安看销售/上下游印证）+ 连续观察。

## 四、操作策略

- 底部重仓（仓位越大越主动）
- 不均衡买入、爆发集中（可达30%+）
- 实业/矿主心态，长期持有不轻易下车
- 组合均衡（不相关行业降波动）
- 行业切换（景气度往上买，变差切）
- 逆向投资（别人悲观时乐观，对国家坚定信心）
- 杠杆慎用（日斗~10%，分红覆盖融资成本；不建议普通人用）

## 五、经典战例（框架验证）

| 标的 | 区间 | 核心逻辑 | 收益 |
|------|------|---------|------|
| 四川长虹 | 1995-1997 | 彩电供不应求，PE 4.5倍 | 10倍 |
| 粤电力B | 1999-2001 | PE 1倍+高分红；B股开放 | 5倍 |
| 伊泰B | 2004-2012 | 煤炭PE<3倍+产能5倍扩张；2012见全行业扩产清仓 | 100倍 |
| 国投电力 | 2013-2015 | 雅砻江水电Capex期结束，利润17倍 | 1倍 |
| 贵州茅台 | 2016-2021 | 八项规定+塑化剂错杀，长期跟踪2016重仓 | 10倍 |
| **大金融** | **2026-** | **PE 5-7倍+PB 1.12倍；"金融是新的消费"范式判断；货币永续扩张+牌照壁垒+头部收敛** | **进行中** |

**共性**：都在行业下行/错杀、供需即将或已经逆转、低估值高分红时布局；在扩产/估值泡沫时退出。

## 六、跨行业供需适配速查（v2.0新增·关键修正）

| 行业类型 | 供需载体 | "头部扩产"红线 | 供需逆转信号 |
|---------|---------|--------------|------------|
| **资源品** | 物理产能（矿井/油田） | ✅ 严格适用 | 行业出清+企业退出+头部不扩产 |
| **制造业** | 产线/装置/船队 | ✅ 严格适用 | 产能出清+政策限产 |
| **金融业** | 资产/负债+牌照 | ❌ 不直接适用 | **头部集中度提升+中小退出+牌照收紧+利差改善** |
| **品牌消费** | 品牌心智+渠道 | ⚠️ 需重新定义 | 品牌出清+渠道收缩+库存去化 |
| **科技/平台** | 用户规模+数据壁垒 | ⚠️ 需重新定义 | 行业出清+用户向头部集中 |

**核心教训**：不可用资源品的供需标准去衡量金融/科技/服务业。金融业的"资本扩张"对应货币增长（永续扩张），不是产能过剩。

## 七、应用误区自检速查（v2.0新增·每次分析前必过）

| 误区 | 表现 | 纠正 |
|------|------|------|
| **五维降维为一维** | 只用"供需关系"下结论 | 五维必须交叉验证，至少三维达标才能建议买入 |
| **供需标准刻舟求剑** | 用资源品标准套金融/科技业 | 先确定行业类型，选择对应供需标准 |
| **静态看动态** | 用当前分红率/ROE做判断 | 看分红潜力和ROE回升空间（动态前瞻） |
| **忽略"有梦想"** | 用行业标签否定想象空间 | "有梦想"是最高维度，需洞察时代变迁 |

**完整自检清单**：见 SKILL.md 第八章。

## 八、与王文原话对照（防走样）

- "任何供求关系发生变化的股票都会涨。" → 分析必谈供需，不纯看估值。
- "周期行业一定要盯住头部公司扩产。" → 卖出首要看扩产。**但注意：此红线针对资源品/制造业，金融业需适配。**
- "分红度一切苦厄。" → 熊市/下行期以分红为安全垫。
- "踏雪寻梅。" → 下行期即调研，不追涨。
- "股票=股权+彩票，我们能把握的就是股权。" → 实业心态，忍波动。
- "知不易，行更难。" → 给结论也要给纪律与耐心预期。
- **"金融是新的消费。"（v2.0新增）** → 金融业是永续成长行业，券商可能是未来的茅台。
- **"金融业其实是一个竞争不那么激烈的行业，而且它的竞争是在收敛的。"（v2.0新增）** → 头部集中度提升=供给收敛。
- **"买金融业股票，某种意义上是把自己口袋里的现金，换成了它账上的现金。"（v2.0新增）** → 金融企业资产即现金。
