# 王文方法论 · 实操分析模板

> 当用户要求"用王文方法论分析某股票/行业"时，按本模板输出结构化报告。重点放在**供需格局与周期位置**判断，这是王文体系区别于纯估值框架的核心。

---

## 分析报告结构（直接套用）

### 标题：`【王文方法论】XX（代码）投资分析`

#### 一、一句话结论
> 例："当前煤炭处于'供不应求'上升通道中段，估值低+分红高，符合王文式建仓区；但需盯住头部扩产信号。"

#### 二、供需格局诊断（核心，最重要）
| 维度 | 当前状态 | 信号强弱 | 说明 |
|------|---------|---------|------|
| **需求端** | 增长/平稳/萎缩 | 🔴🟡🟢 | 用电量/下游产量/消费趋势 |
| **供应端（盯紧）** | 收缩/平稳/扩张 | 🔴🟡🟢 | 头部公司扩产？资本开支？政策限产？ |
| **库存/价格** | 去化/平稳/累积 | 🔴🟡🟢 | 价格是否触底回升 |
| **供需结论** | 供过于求 / 紧平衡 / 供不应求 | — | 综合判断 |

**关键追问（按传导链往上找因）**：
1. 头部公司近1-2年资本开支/扩产计划如何？（扩产=未来危险）
2. 行业是否经历了长期下行、企业退出、供给自然收缩？
3. 有无政策/资源硬约束限制新供应？（安监、能耗、环保、矿权）
4. 外源变故（如碳中和、关税）是否**改变**了供需关系？→ 更值钱还是更不值钱？

#### 三、周期位置定位（四大周期）
- **经济/货币周期**：宽松/中性/收紧
- **行业周期**：上行/筑底/下行（结合供需结论）
- **企业/产业周期**：Capex见顶利润释放 / 扩产利润压缩 / 平稳
- **人性/情绪周期**：贪婪（顶）/ 恐慌（底）/ 中性
- **综合判断**：处于"第一笔买入区 / 集中加仓区 / 持有区 / 警惕卖出区"

#### 四、五维体检
| 维度 | 是否满足 | 数据/证据 |
|------|---------|----------|
| 低估值 | ✅/❌ | PE/PB分位数、同业对比 |
| 高现金流 | ✅/❌ | 经营现金流/净利润、应收账款 |
| 高分红 | ✅/❌ | 股息率、分红率、vs国债 |
| 业务可持续 | ✅/❌ | 是否减量市场、时代变迁风险 |
| 有梦想（预期差） | ✅/❌ | 市场未认知的改变、弹性来源 |

#### 五、操作策略（分批与纪律）
- **当前动作**：第一笔买入 / 集中加仓 / 持有待涨 / 警惕减仓
- **仓位建议**：如"建首仓20%，供需进一步确认且价格未暴涨加至50%，头部扩产信号出现则止盈"
- **卖出纪律**：①头部扩产 ②外源变故改变供需 ③找到更优标的
- **持有心态**：实业/矿主心态，以分红为熊市防线，不轻易下车

#### 六、风险提示
- 哪个外源因素可能改变供需关系？（列明并给监控指标）
- 周期长度风险：可能买早，需用分红+耐心熬

#### 七、与王文经典案例对标
> 例："本标的类似王文2012年前的伊泰B（低PE+产能扩张预期），但需警惕其扩产节奏不同于当年。"

## 输出风格要求
1. **数据→业务→行业→风险→估值** 的结构化顺序（贴合用户习惯）。
2. 多用**表格对比**，少用长篇空话。
3. 明确给出**观点与买卖区间**，不模棱两可。
4. 强调**供需先行**，估值次之。
5. 如与用户"内生回报投资法"结合，点明互补处（如分红回报重合、周期拐点互补）。
