> **v2.0关键更新**：新增"第〇步：行业类型识别与供需标准适配"，避免将资源品供需标准机械套用到金融/消费/科技行业。新增"五维交叉验证自检"和"范式判断"模块。

## 分析报告结构（直接套用）

### 标题：`【王文方法论】XX（代码）投资分析`

#### 〇、行业类型识别与供需标准适配（v2.0新增·必做第一步）

> **在做任何供需判断之前，必须先确定行业类型并选择对应的供需标准。** 不同行业的供需载体和"头部扩产"红线的含义完全不同。

| 识别项 | 选择 | 说明 |
|--------|------|------|
| **行业类型** | ☐ 资源品 ☐ 制造业 ☐ 金融业 ☐ 品牌消费 ☐ 科技/平台 | 决定使用哪种供需标准 |
| **供需载体** | 物理产能 / 产线装置 / 资产+牌照 / 品牌+渠道 / 用户+数据 | 该行业的"供给"到底是什么 |
| **"头部扩产"红线是否适用** | ☐ 严格适用 ☐ 不直接适用 ☐ 需重新定义 | 资源品=严格适用；金融业=不直接适用（资本扩张≠产能过剩） |
| **表面供需 vs 实质供需** | 表面：企业数量/费率 → 实质：集中度趋势/壁垒变化 | 如券商40+家是表面过剩，但头部集中度提升=实质收敛 |

> ⚠️ **如果行业类型为金融业，请同时加载 `financial_sector_guide.md` 并使用金融业适配标准。**

#### 一、一句话结论
> 例："当前煤炭处于'供不应求'上升通道中段，估值低+分红高，符合王文式建仓区；但需盯住头部扩产信号。"
> 例（金融业）："券商PE 5-7倍极端低估+头部集中度提升+货币扩张驱动需求永续增长，符合王文式建仓区。"

#### 二、供需格局诊断（核心，最重要）

> **注意**：以下表格中的"供应端"判断必须使用第〇步确定的行业适配标准。

| 维度 | 当前状态 | 信号强弱 | 说明 |
|------|---------|---------|------|
| **需求端** | 增长/平稳/萎缩 | 🔴🟡🟢 | 资源品：用电量/下游产量；金融业：货币增速/成交额/财富管理需求 |
| **供应端（盯紧）** | 收缩/平稳/扩张 | 🔴🟡🟢 | 资源品：头部扩产/资本开支/政策限产；金融业：头部集中度/牌照壁垒/中小机构退出 |
| **价格/利差信号** | 触底回升/平稳/下行 | 🔴🟡🟢 | 资源品：产品价格；金融业：佣金率/利差结构/ROE |
| **供需结论** | 供过于求 / 紧平衡 / 供不应求 | — | 综合判断（注意区分表面供需与实质供需） |

**关键追问（按传导链往上找因）**：
1. 头部公司近1-2年资本开支/扩产计划如何？
   - 资源品：扩产=未来危险（卖出红线1）
   - 金融业：资本扩张≠产能过剩，需看资本回报率是否提升
2. 行业是否经历了长期下行、企业退出、供给自然收缩？
3. 有无政策/资源硬约束限制新供应？（资源品：安监/环保/矿权；金融业：牌照/监管/资本要求）
4. 外源变故是否**改变**了供需关系？→ 更值钱还是更不值钱？
5. **头部集中度趋势如何？**（v2.0新增）集中度提升=供给收敛（利好）；参与者增多=供给扩张（需警惕）

#### 三、周期位置定位（四大周期）
- **经济/货币周期**：宽松/中性/收紧
- **行业周期**：上行/筑底/下行（结合供需结论）
- **企业/产业周期**：Capex见顶利润释放 / 扩产利润压缩 / 平稳
- **人性/情绪周期**：贪婪（顶）/ 恐慌（底）/ 中性
- **综合判断**：处于"第一笔买入区 / 集中加仓区 / 持有区 / 警惕卖出区"

#### 四、五维体检（必须交叉验证，不可单维下结论）

> ⚠️ **v2.0自检**：五维必须全部检验后才能给出结论。至少三个维度达标才能建议"买入"。如果只有一维不达标但其余四维强支撑，需分析该维度是否使用了正确的行业标准。

| 维度 | 是否满足 | 数据/证据 | 判断标准说明 |
|------|---------|----------|------------|
| 低估值 | ✅/❌ | PE/PB分位数、同业对比 | 所有行业同标准 |
| 高现金流 | ✅/❌ | 经营现金流/净利润、应收账款 | 金融业：资产本身是现金/货币化资产 |
| 高分红 | ✅/❌ | 股息率、分红率、vs国债 | ⚠️ **动态看**：当前分红率 + 利润释放后的提升空间 |
| 业务可持续 | ✅/❌ | 是否减量市场、时代变迁风险 | 金融业：货币扩张=永续成长 |
| 有梦想（预期差） | ✅/❌ | 市场未认知的改变、弹性来源 | ⚠️ **最高维度**：是否正在发生范式级时代变迁？ |

#### 四-补、范式判断（v2.0新增·"有梦想"的最高形态）

> 该行业是否正在发生未被市场认知的**范式级变迁**？这是王文方法论中"有梦想"维度的最高应用。

| 判断维度 | 内容 |
|---------|------|
| 时代变迁 | 是否有宏观层面的结构性变迁（如人口老龄化→金融需求爆发） |
| 范式重定义 | 该行业是否正在被重新定义（如"金融是新的消费"） |
| 长期空间 | 5-10年后的行业规模是否远大于当前（如交易量5年后4万亿） |
| 预期差强度 | 市场当前定价与上述变迁之间的差距有多大 |

#### 五、操作策略（分批与纪律）
- **当前动作**：第一笔买入 / 集中加仓 / 持有待涨 / 警惕减仓
- **仓位建议**：如"建首仓20%，供需进一步确认且价格未暴涨加至50%，头部扩产信号出现则止盈"
- **卖出纪律**：
  - 资源品：①头部扩产 ②外源变故改变供需 ③找到更优标的
  - 金融业：①利差结构恶化/坏账激增 ②监管根本性转向 ③找到更优标的
- **持有心态**：实业/矿主心态，以分红为熊市防线，不轻易下车

#### 六、风险提示
- 哪个外源因素可能改变供需关系？（列明并给监控指标）
- 周期长度风险：可能买早，需用分红+耐心熬
- **行业适配风险**（v2.0新增）：是否使用了错误的供需标准判断该行业？

#### 七、与王文经典案例对标
> 例："本标的类似王文2012年前的伊泰B（低PE+产能扩张预期），但需警惕其扩产节奏不同于当年。"
> 例（金融业）："本标的类似王文2026年重仓金融的逻辑（极端低估+头部收敛+货币永续增长）。"

#### 八、五维交叉验证自检（v2.0新增·出结论前必过）

```
□ 1. 我是否在第〇步确定了行业类型并选择了对应的供需标准？
□ 2. 我是否五个维度全部检验了？（不能只靠供需一维下结论）
□ 3. 我对分红的判断是静态的还是动态的？（看分红潜力，不仅看当前分红率）
□ 4. 我是否考虑了时代变迁/范式判断？（"有梦想"维度）
□ 5. "头部扩产"红线在本行业是否适用？（金融业不直接适用）
□ 6. 我判断的"供过于求"是表面的还是实质的？（企业数量多≠供过于求，看集中度趋势）
□ 7. 我的结论是否与行业领军人物的操作方向一致？如不一致，是哪里认知不同？
```

## 输出风格要求
1. **数据→业务→行业→风险→估值** 的结构化顺序（贴合用户习惯）。
2. 多用**表格对比**，少用长篇空话。
3. 明确给出**观点与买卖区间**，不模棱两可。
4. 强调**供需先行**，估值次之。
5. **五维交叉验证**后才能下结论，不可用单一维度替代全部判断。
6. 如与用户"内生回报投资法"结合，点明互补处（如分红回报重合、周期拐点互补）。
