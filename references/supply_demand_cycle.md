# 供求关系判断行业周期 · 详细方法（王文方法论核心）

> 本文件是"王文投资方法论"技能的核心模块，专门讲清**如何通过供求关系判断行业处于上升通道还是下行通道、拐点何时出现**。所有要点均提炼自《万倍叔：日斗投资王文的财富心得》。

---

## 一、总纲：供需关系是商业的"元规则"

王文原话：
> "商业的元规则，第一规则是供需关系，一切服从供需关系。……任何供求关系发生变化的股票都会涨。任何一个（行业），因为不能用想象。股票分为认知的部分和修行的部分，认知的部分就是：真实的世界到底是什么？你越接近于事实，就越接近成功。"

**关键推论**：
- 不要从价格涨跌倒推"行业好不好"，而要从**供需结构**正推价格。
- 任何行业，只要供求关系发生**逆转**（供过于求→供不应求），股价迟早涨；反之亦然。
- 大众**普遍低估供需变化的周期长度**（能达8年以上），所以"买早了还在跌"是正常的，不是判断错。

## 二、逻辑传导链：从结果反推"因"

```
股价（最滞后，结果）
  ← 产品价格（煤价/钢价/运价/锂价）
    ← 供求关系（供过于求 ↔ 供不应求）
      ← 供应端的资本行为【核心：盯供应】
        ← 生产者态度 + 国家态度 + 资本开支（投资）
```

王文以煤炭为例：
> "煤炭特别受煤价影响，煤价在股价之前；在煤价之前是库存和供求关系；在供求关系之前，是生产者的态度、国家的态度、还有我们的投资。越往上找得准、找得及时，把握越犀利。"

**操作含义**：
- 看股价 → 太晚（已反映）
- 看煤价/产品价格 → 晚一点
- 看库存 + 供需 → 及时
- 看**头部公司资本开支、扩产计划、国家政策（安监/发改委）、行业盈亏** → 最早、最准

## 三、买入信号：供求关系"逆转"为供不应求

### 供不应求的充要条件
| 要件 | 表现 | 监测指标 |
|------|------|---------|
| **需求持续增长** | 用电量、下游产量、消费稳健或上行 | 行业下游需求增速、宏观景气 |
| **无新供应** | 资本进不来、无新产能投产、存量收缩 | ① 行业长期亏损→企业退出 ② 头部公司高分红不扩产 ③ 无新批产能/新矿投产 ④ 政策限产（安监、能耗、环保） |

### 何时下手
- **第一笔**：确认"供求关系已逆转、供不应求"即买（不等内容完全price in）。
- **集中仓**：待行业爆发（量价齐升、业绩兑现）时把该行业仓位加到组合30%+。
- 王文："我们闻到隆隆的炮声时，要勇敢前进，上帝给你的机会有限。"

### 反面：为什么"便宜"可能是陷阱
> "有些人买了底部但下面还有地下十八层，就是还没看明白供求关系，供求关系还没发生变化。"

→ 估值低 ≠ 拐点已到。**必须确认供需逆转，而非单纯便宜。**

## 四、卖出信号：三条红线

### 红线1：头部公司纷纷大规模扩产（最危险）
> "周期行业一定要盯住头部公司扩产。如果大家纷纷上产能，那就是卖出的信号。……资本无序进来→过度投资→过度扩产→产能过剩→价格暴跌→股价暴跌。产品价格下跌带来股价暴跌，这是链条。你关注最后的价格只是结果，因在于资本进来。"

- 实战：王文2004-2012持伊泰B，2012年看到"每家头部煤炭企业都在扩产"，利润仍好即全清仓，卖在顶部。
- 适用所有周期行业：煤炭、锂、面板、硅料、航运……扩产快=危险来得快（制造业扩产比资源品更快）。

### 红线2：外源变故改变供需关系
> "判断行业碰到变故，是否改变供需关系。比如煤炭，煤炭达峰我认为不会改变供需关系，就加仓；若发现变故要改变供需关系，就卖。一切外源因素必须落到：让公司更赚钱还是不赚钱？更值钱还是不值钱？让公司更值钱就买，导致不值钱就卖。"

- 极简决策树：**外源冲击 → 影响供需？ → 公司更值钱(买/持有) or 更不值钱(卖)**

### 红线3：找到明显更好的标的
> "我过去卖股票最重要的理由，是我找到了一个更好的。把那些也好的、差不多的、还行的全卖了，集中到最好的，连犯错误的机会都没有。"

## 五、周期长度与底部陷阱（心理准备）

| 现象 | 说明 | 应对 |
|------|------|------|
| **周期可达8年+** | 供需变化周期远长于多数人预期 | 买早了不慌，用分红熬 |
| **下行期特别长** | 行业内人长期亏钱→不断退出→供求才运转 | 下行末端才是布局期 |
| **底部有"十八层"** | 看似底但实际供需未逆转 | 必须确证供需逆转才重仓 |
| **上涨不均衡** | 第一个50%等三年，第二个50%半年，第三个50%三周，第四个50%一周 | 反转即买，爆发集中加 |

煤炭完整周期示意：
```
2004 ──涨──► 2012（见全行业扩产，清仓）
2012 ──跌──► 2020（8年调整，企业退出、供给收缩）
2020 ──供不应求再涨──► （无新矿投产+需求增长）
```

## 六、四大周期叠加判断框架

> 股票市场 = 经济周期 + 人性周期 + 资金周期 + 企业成长周期(产业周期) 的叠加。

| 周期 | 判断要点 | 当前信号举例 |
|------|---------|------------|
| **经济/货币周期** | 货币宽松涨、收紧跌；监管周期友好度 | 利率下行、监管友好→向上 |
| **行业周期** | 景气度方向；上下游轮动（先煤后电） | 行业供需逆转→向上 |
| **企业/产业周期** | 资本开支期结束=利润释放；扩产=利润压缩 |  Capex见顶→利润释放 |
| **人性/情绪周期** | 底顶易判：全民谈股=顶；90%亏钱=底 | 利用情绪，不被动 |

使用原则：
- **宏观与微观矛盾时**：价值投资者永远买好公司，但要在市场低迷时多信"好的"、高涨时多信"坏的"。
- **七八年一轮回**：A股历史上约七八年一个周期，把握周期事半功倍。

## 七、调研与验证：踏雪寻梅 + 交叉验证

### 踏雪寻梅（时机）
- 行业**下行周期**就开始关注（不是涨了才追）。
- 产业链调研、建立人脉、听不同人讲。
- 长期高强度、高频关注，直到确信机会出现。

### 交叉验证（防误导）
| 对象 | 问什么 | 警惕 |
|------|--------|------|
| 董事长 | 愿景、规划 | 警惕画饼 |
| 工程师/技术 | 技术真实性 | — |
| 门口保安/销售 | 实际销售情况 | 往往比高管更靠谱 |
| 上下游/竞争对手 | 客观评价 | 相互印证 |

### 连续观察
> "认真读年报，连续观察公司说的做到了没有，不断修正判断。投资是连续观察、不断矫正的过程。"

## 八、30秒速查清单（判断行业拐点用）

回答以下问题，YES越多，越接近"供求关系逆转、可建仓"：

- [ ] 行业是否经历了**长期（3-8年）下行/亏损**，企业不断退出？
- [ ] 头部公司是否**高分红、不扩产、资本开支低迷**？
- [ ] 是否有**政策/资源约束**限制新供应（安监、能耗、环保、无新批产能）？
- [ ] **需求端**是否稳健或增长（非断崖）？
- [ ] 库存/价格是否出现**触底回升**的早期信号？
- [ ] 是否**无新增大产能**即将投产？

→ 若以上多为YES，且估值低、分红高：这是王文式"供不应求"建仓区。
→ 若出现"头部公司纷纷扩产/资本大规模涌入"：警惕，准备减仓。

反之，若出现以下，则是**卖出/回避**信号：
- [ ] 头部公司集体公告大规模扩产？
- [ ] 行业遭遇改变供需的外源变故（让公司更不值钱）？
- [ ] 找到明显更优的替代标的？
