# 王文投资方法论（日斗投资）

> 源自《万倍叔：日斗投资王文的财富心得》。王文，1993年入市，30年实现万倍收益，掌舵百亿私募"日斗投资"。其方法论的本质是：**用实业投资的眼光，在供求关系发生逆转的拐点重仓高分红垄断型资产，靠长期持有与分红穿越周期。**

## When to Use（触发场景）

加载本技能当用户：
- 要求"用王文方法论 / 日斗投资思路 / 供求关系分析"评估某股票、行业或标的
- 询问**如何判断行业周期拐点、上升/下行通道、供需格局是否逆转**
- 分析**周期股（煤炭、石油、有色、航运、面板等）**的投资时机
- 评估**高分红、低估值、高现金流**类型的"现金奶牛"资产
- 需要**选股框架、买卖点纪律、仓位管理**方面的结构化建议
- 提及"踏雪寻梅""分红度一切苦厄""成住坏空""两高一低+有梦想"等王文术语

## When NOT to Use

- 纯量化短线交易、技术指标（MACD/K线形态）分析——本体系是基本面长周期框架
- 纯宏观叙事而无具体标的——需落到"某行业/某公司的供需与估值"
- 与用户既有"内生回报投资法"冲突时：本技能作为**补充视角**（供需周期判断），应与用户自有框架融合，不替代。

---

## 一、核心思想（道）：三大铁律

1. **供需关系是商业的元规则（第一规则）**："一切服从供需关系。任何供求关系发生变化的股票都会涨。" 这是王文方法论的总纲，比估值、比财务报表都更底层。
2. **垄断是最核心的价值**：自然垄断（煤炭）、资源/品牌垄断（茅台）、牌照垄断（平安）。垄断能对抗通胀、产品能涨价，单价涨×销量增=利润增=股东分到的钱增。
3. **股票=股权+彩票，能把握的只有股权**：以"矿主心态"做投资，忍受股价波动是获取高于债券收益所支付的成本。

> **投资分两个层面**：认知（识别优秀企业、竞争优势、盈利能力）+ 修行（在无数下跌日子里保持平静乐观）。认知靠学习，修行靠定力。

---

## 二、五维选股标准：两高一低可持续 + 有梦想

| 维度 | 内核 | 操作含义 |
|------|------|---------|
| **低估值** | 估值给的是"商业模式"，不同模式不同PE | 煤炭/银行10倍PE可能贵，茅台25倍PE可能便宜。行业估值跌到下沿/跌破下轨才考虑买 |
| **高现金流** | 现金流反映真实经营成果 | 凡"星辰大海"但现金流为负，基本不投。只投生命周期到了能分享经营成果的阶段（风投的事不干） |
| **高分红（中心）** | 分红穿透三张表：利润表赚钱+现金流充足+资产负债健康；也检验管理层人品（愿分享） | 熊市最后防线——"分红度一切苦厄"。分红比国债高，资金迟早会反应 |
| **业务长期可持续** | 警惕时代变迁与减量市场 | 钢铁虽符合前三点但业务不可持续，不投 |
| **有梦想** | 前四点保证不亏大钱，此点是超额收益来源 | 行业已发生市场未认知的改变（预期差），一旦认知到就有巨大涨幅 |

**关键提醒**：前四点（低估值/高现金流/高分红/可持续）保证"买的每笔钱都不输"；"有梦想"是弹性来源。便宜可能是陷阱——**若行业处于下行周期，这种便宜是价值陷阱，须严控仓位**。

## 三、供求关系判断行业周期的方法（术·核心）

> 这是本技能最具操作价值的部分。王文判断行业周期不靠宏观预测，而靠**顺藤摸瓜找"因"**。

### 3.1 逻辑传导链（从结果反推原因）

```
股价（结果，最滞后）
  ← 价格（煤价/钢价/运价）
    ← 供求关系（供过于求 / 供不应求）
      ← 供应端的资本行为（核心盯供应）
        ← 生产者的态度 + 国家的态度 + 投资（资本开支）
```

**王文原话**："煤价在股价之前，煤价之前是库存和供求关系，供求关系之前是生产者的态度、国家的态度、我们的投资。越往上找越准、越及时。"

**核心口诀**：**盯供应，不盯价格。** 价格只是结果，因在"资本无序进入→过度投资→过度扩产→产能过剩→价格暴跌→股价暴跌"这一链条的起点。

### 3.2 买入信号：供求关系逆转（供过于求 → 供不应求）

- **供不应求的实质** = 需求持续增长 + 无新供应（资本进不来 / 无新矿投产）
- **资本进不来的标志** = 行业长期亏钱→企业不断退出→供给收缩；同时头部公司高分红、不扩产、没人新进投资
- **操作**：在供求关系发生逆转、即"供不应求"时买入第一笔；待行业爆发时集中仓位（可加到组合30%+）

### 3.3 卖出信号（三条）

1. **头部公司纷纷大规模扩产** = 最危险的信号。王文2004-2012持伊泰B，2012年看到"每家头部企业都在扩产"、利润仍好就全清仓，卖在顶部。
2. **行业遭遇重大变故，且该变故改变供需关系** → 卖。王文原话："判断变故是否改变供需关系。煤炭达峰若不改变供需就加仓；若改变就卖。" 标准极简：**外源因素让公司更值钱就买，更不值钱就卖。**
3. **找到明显更好的标的** → 换仓集中到最好（"连犯错误机会都没有"）

### 3.4 周期长度的残酷性（最易被低估）

- 供需变化的周期**能长达8年以上**。王文："很多人一买就跌，是因为低估了周期的长度和残酷度。"
- 煤炭案例：2004-2012上涨（8年）→ 2012-2020下跌（8年，全行业调整）→ 2020年后供不应求再涨。
- **下行时间长的原因**：行业内人长期亏钱、不断退出，供求才得以运转。所以"向下的时间会非常长"。
- **底部陷阱**："很多人买了底部，下面还有地下十八层，是因为还没看明白供求关系，供求关系还没发生变化。"

### 3.5 四大周期框架

| 周期 | 内涵 | 用法 |
|------|------|------|
| **经济/货币周期** | 货币多则股市涨，收货币则跌 | 看利率、流动性、监管周期 |
| **行业周期** | 先煤炭后电力（煤涨电亏，煤不行电利润好）；行业向上时买 | 行业轮动，景气度切换 |
| **企业/产业周期** | 资本开支期结束=利润释放（如水电） | 跟踪产能投放节奏 |
| **人性/情绪周期** | 底顶易判：所有人谈股票=高点；90%亏钱=低点 | 利用情绪，不被控制 |

### 3.6 调研方法：踏雪寻梅 + 交叉验证

- **踏雪寻梅**：行业下行周期就开始关注，而非等涨了才看。产业链调研、建人脉、长期高频关注，直到确信机会出现。
- **交叉验证**（避免被上市公司误导）：
  - 问董事长 → 看愿景（警惕画饼）
  - 问工程师 → 看技术真实性
  - 问门口保安 → 看销售情况（往往比问高管更靠谱）
  - 关注上下游与竞争对手评价
- **连续观察**：看公司说的做到了没有，不断修正判断（"投资是连续观察、不断矫正的过程"）

## 四、操作策略（术·执行）

1. **底部重仓**："在底部建立大仓位，底部仓位越大越主动，未来亏钱机会越少。"
2. **不均衡买入**：股票上涨不均衡——第一个50%可能等三年，第二个50%半年，第三个50%三周，第四个50%一周。故**发现反转即买，爆发时集中仓位**。
3. **实业/矿主心态**：把股价波动当成本，长期持有不轻易下车（"当时轻别意中人，山高水长无觅处"）。高分红3-8年回本即无压力。
4. **组合均衡**：投资不相关行业（白酒/运营商/家电/银行/煤炭）降低单一行业波动。
5. **行业切换**：景气度往上走时买；变差则切换（牛市此起彼伏涨，熊市此起彼伏跌）。
6. **逆向投资**：别人悲观时乐观，对国家发展坚定信心；五次抄底。"悲观者永远正确，乐观者才能赚到钱。"
7. **杠杆（慎用）**：日斗维持约10%杠杆，分红（6%）覆盖融资成本才安全；**不建议普通投资者用**。

## 五、经典战例（验证框架）

| 标的 | 时间 | 逻辑（供需/估值） | 收益 |
|------|------|------|------|
| 四川长虹 | 1995-1997 | 彩电严重供不应求，PE 4.5倍 | 10倍 |
| 粤电力B | 1999-2001 | 市值9亿/年利8.35亿，PE 1倍，分红可观；政策开放B股 | 5倍 |
| 伊泰B | 2004-2012 | 煤炭，PE<3倍，产能5倍扩张空间；2012见全行业扩产清仓 | 100倍 |
| 国投电力 | 2013-2015 | 雅砻江水电资本开支期结束，电价确定、煤价可能跌 | 1倍（利润17倍） |
| 贵州茅台 | 2016-2021 | 八项规定+塑化剂错杀，长期跟踪，2016重仓 | 10倍 |

## 六、使用方法（本技能如何落地）

当用户给出具体标的/行业时，按 `references/analysis_template.md` 的模板输出结构化分析，重点回答：

1. **供需格局**：当前供过于求还是供不应求？需求端趋势？供应端（头部公司扩产/资本开支/政策）信号？
2. **周期位置**：处于四大周期（经济/行业/企业/情绪）的哪一阶段？拐点信号出现了吗？
3. **五维体检**：低估值？高现金流？高分红？可持续？有梦想（预期差）？
4. **买卖决策**：当前是"第一笔买入"还是"集中加仓"还是"警惕卖出（头部扩产）"？给分批仓位建议。
5. **风险提示**：哪个外源变故可能改变供需关系？

> 详细方法论、供需判断检查清单、分析模板见 `references/` 目录。

## 与其他框架的关系

- 与用户"内生回报投资法"互补：王文侧重**行业供需周期拐点**（买在周期反转），内生回报侧重**分红-价值与成长-反转双模型**（买在便宜）。二者在"低估值+高分红+长期持有"上高度一致。
- 与巴菲特/段永平一脉相承：垄断、能力圈、对下跌高兴、长期主义。王文的中国特色在于**多重博弈市场的分红防御 + 供求关系元规则**。
